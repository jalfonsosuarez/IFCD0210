Tipos de datos y operadores en Javascript

Primitivos

  • number: x = 5
  • string: x = "5"
  • boolean: x = false

Operadores aritméticos

  • Suma: + (2 + 2=4)
  • Multiplicación: * (2 * 2 = 4)
  • División: / (4 / 2 = 2)
  • Esponenciacion: ** (2 ** 2 = 8)
  • Modulo: % (100 % 2 = 0) (9 % 2 = 1)

Precedencia de operadores

  • 1. Paréntesis ()
  • 2. Multiplicación, división, módulo * / %
  • 3. Suma y resta + -
  • 4. Operadores lógicos AND, OR, NOT

Cadenas de texto

  • Comillas simples o dobles. 'Es interesante la comilla ""'
  • Operador de contatenación: + ("cadena1 + cadena2")

Boolean

  • true -> 1 ó "cadena con contenido"
  • false -> 0, cualquier valor numérico ó ""

Operadores lógicos

  • && -> and
  • || -> or
  • ! -> not
  • ^ -> xor

Operadores de comparación

  • Igualdad: == , ===
  • Desigual: !==, !=
  • Mayor que: >
  • Mayor o igual que: >=
  • Menor que: <
  • Menor o igual que: <=
