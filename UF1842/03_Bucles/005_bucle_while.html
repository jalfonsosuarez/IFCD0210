Bucle while en Javascript

-> Estructura del bucle while:


        while (condicion) {
            bloque de instrucciones
            control de la condicion
            continue;
            break;
        }

Para ver el resultado de la ejecución del código, abre la consola del inspector.

-> Imprimir los primeros 20 numeros:


        let vueltas = 0;
        while (vueltas < 20) {
            console.log("Vuelta: " + vueltas);
            vueltas++;
        }

-> Bucle while infinito con salida controlada por if:

    
        let vueltas = 0;
        while (true) {
            vueltas++;
            console.log("Vuelta: " + vueltas);
            if (vueltas === 20) break;
        }

-> Bucle while infinito con salida controlada por cambio de estado en la condición:

    
        let vueltas = 0;
        let terminar = false;
        while (!terminar) {
            vueltas++;
            console.log("Vuelta: " + vueltas);
            if (vueltas === 20) terminar = true;
        }

-> Bucle while, cuenta regresiva de 10 a 0:

    
        let n = 10;
        while (n >= 0) {
            console.log(n);
            n--;
        }

-> Bucle while, uso de continue, sacar los 20 primeros pares:

    
        let n = -1;
        while ( n < 20 ) {
            n++;
            if ( n % 2 !== 0) {
                continue;
            }
            console.log(n);
        }
