Arrays en Javascript

Más información sobre arrays.

-> Definición de un array

        
            -> Un array es un tipo especial de objeto de Javascript.
        
            let miArray = [ elemento_1, elemento_2, ....];

-> Longitud de un array:

        
            let miArray = [ "Uno", "Dos", "Tres", 5, 6, 20];
            console.log(miArray.length);
        
         longitudArray();

-> Primer elemento de un array:

        
            let miArray = [ "Uno", "Dos", "Tres", 5, 6, 20];
            console.log(miArray[0]);
        
         primerElementoArray();

-> Último elemento de un array:

        
            let miArray = [ "Uno", "Dos", "Tres", 5, 6, 20];
            console.log(miArray[miArray.length - 1]);
        
         ultimoElementoArray();

-> Añadir un elemento al final de un array:

        
            let miArray = [ "Uno", "Dos", "Tres", 5, 6, 20];
            console.log(miArray);
            miArray.push('Nuevo elemento');
            console.log(miArray);
        
         sumarElementoArray();

-> Usando el operador spread (propagación) en un array:

        
            let miArray = [ "Uno", "Dos", "Tres", 5, 6, 20];
            let prop = [...miArray, 'más elementos'];
            console.log(prop);
        
         propArray();

-> Usando el operador spread (propagación) en un array y una cadena:

        
            let miCadena = "Hola mundo";
            let prop = [...miCadena];
            console.log(prop);
        
         propStringArray();

-> Array de letras a string::

        
            function arrayToString(array) {
                let cadena = '';
                for (n=0; n < array.lenght; n++) {
                    cadena += array[n];
                }           
                console.log(cadena);
            }
        
         arrayToString(['A','B','C',' ','D','E']);
