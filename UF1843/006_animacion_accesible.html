Acordeón accesible con animación

¿Cómo funciona la animación?

Usamos CSS Grid para animar la apertura sin depender de height: auto, que no es animable.

¿Esto afecta la accesibilidad?
  • El HTML sigue siendo semántico
  • El foco y teclado funcionan igual
  • Los lectores de pantalla anuncian el estado
¿Qué pasa si el usuario reduce animaciones?

Detectamos prefers-reduced-motion y desactivamos las transiciones automáticamente.
